ขอบข่ายงาน

บทบาทที่เปิดรับ 4 ตำแหน่ง

แต่ละตำแหน่งมีขอบข่ายงานและอัตราค่าบริการเริ่มต้นต่างกัน ตามระดับใบประกอบวิชาชีพ — เนื้อหาด้านล่างเป็นกรอบเบื้องต้น ทีมงาน CMCC จะยืนยันรายละเอียดเฉพาะเคสอีกครั้งก่อนมอบหมายงาน

เริ่มต้น 700 บาท / ครั้ง

นักกายภาพบำบัด (ใบประกอบวิชาชีพ)

  • ประเมินร่างกายและวางแผนโปรแกรมฟื้นฟูเฉพาะราย
  • ฝึกเคลื่อนไหว การทรงตัว และการเดิน (Gait training)
  • ฟื้นฟูผู้ป่วยโรคหลอดเลือดสมอง กระดูก-ข้อ และผู้สูงอายุติดบ้าน/ติดเตียง
  • สอนญาติ/ผู้ดูแลให้ฝึกต่อได้อย่างถูกวิธี
  • บันทึกความก้าวหน้าและรายงานผลกลับทีม CMCC

ต้องมีใบประกอบวิชาชีพกายภาพบำบัดที่ยังไม่หมดอายุ

เริ่มต้น 700 บาท / ครั้ง

นักกิจกรรมบำบัด (ใบประกอบวิชาชีพ)

  • ประเมินความสามารถในการทำกิจวัตรประจำวัน (ADL)
  • ฝึกทักษะการใช้มือ การรับรู้ และความจำในผู้ป่วยสมอง
  • ปรับสภาพแวดล้อมบ้านให้เหมาะกับข้อจำกัดของผู้ป่วย
  • ออกแบบกิจกรรมบำบัดเฉพาะบุคคลตามเป้าหมายครอบครัว
  • บันทึกความก้าวหน้าและรายงานผลกลับทีม CMCC

ต้องมีใบประกอบวิชาชีพกิจกรรมบำบัดที่ยังไม่หมดอายุ

เริ่มต้น 500 บาท / ครั้ง

ผู้ช่วยนักกายภาพบำบัด

  • ช่วยฝึกโปรแกรมที่นักกายภาพบำบัดออกแบบไว้ล่วงหน้า
  • ดูแลการออกกำลังกายซ้ำตามรอบที่กำหนด
  • ช่วยพยุงเดิน จัดท่าทาง และดูแลความปลอดภัยขณะฝึก
  • รายงานอาการผิดปกติกลับนักกายภาพบำบัดผู้ดูแลเคส

ทำงานภายใต้การควบคุมของนักกายภาพบำบัดประจำเคสเสมอ

เริ่มต้น 500 บาท / ครั้ง

ผู้ช่วยนักกิจกรรมบำบัด

  • ช่วยดำเนินกิจกรรมบำบัดตามแผนที่นักกิจกรรมบำบัดวางไว้
  • สนับสนุนการฝึกกิจวัตรประจำวันร่วมกับผู้ป่วยและญาติ
  • เตรียมอุปกรณ์และจัดสภาพแวดล้อมสำหรับกิจกรรม
  • รายงานความคืบหน้ากลับนักกิจกรรมบำบัดผู้ดูแลเคส

ทำงานภายใต้การควบคุมของนักกิจกรรมบำบัดประจำเคสเสมอ

ค่าบริการ

อัตราเริ่มต้นตามระดับวิชาชีพ

ค่าบริการต่อครั้งอาจปรับตามระยะเดินทาง ความซับซ้อนของเคส และระยะเวลาการฝึก โดยจะแจ้งครอบครัวให้ทราบก่อนยืนยันนัดหมายทุกครั้ง

500บาท / ครั้ง เริ่มต้น

เหมาะสำหรับเคสที่มีแผนฟื้นฟูจากนักวิชาชีพแล้ว ต้องการผู้ช่วยฝึกต่อเนื่องตามรอบ

นักกายภาพบำบัด / นักกิจกรรมบำบัด
700บาท / ครั้ง เริ่มต้น

เหมาะสำหรับการประเมินครั้งแรก วางแผนโปรแกรม หรือเคสที่ต้องใช้วิจารณญาณทางวิชาชีพ

การเตรียมความพร้อม

ทุกคนฝึกที่ศูนย์ CMCC ก่อน 2–3 วัน ก่อนออกเคสจริง

เพื่อให้มาตรฐานการดูแลของพาร์ทเนอร์ทุกคนสอดคล้องกัน ก่อนเริ่มรับเคส นักวิชาชีพและผู้ช่วยทุกท่านจะเข้าร่วมการปฐมนิเทศและฝึกปฏิบัติที่ศูนย์

วันที่ 1

ปฐมนิเทศและมาตรฐานงาน

ทำความเข้าใจแนวทางของ CMCC ระบบรายงานเคส จรรยาบรรณ และความปลอดภัยขณะปฏิบัติงานที่บ้านผู้ป่วย

วันที่ 2

ฝึกปฏิบัติหน้างานจริง

ฝึกทักษะการประเมิน การพยุง การจัดท่า และการสื่อสารกับครอบครัวผู้ป่วย ภายใต้การดูแลของพี่เลี้ยง

วันที่ 3

ประเมินความพร้อมและมอบหมายเคส

ประเมินผลการฝึก ให้คำแนะนำเพิ่มเติมรายบุคคล และจับคู่กับเคสแรกตามพื้นที่และความถนัด
